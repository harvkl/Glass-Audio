# ![Glass Audio](resources/media/Icon.png) Glass Audio - an audio player that receives and manages your Windows media.

A desktop C# (WPF) app for managing Windows media-sessions and system volume. Allows you to control playback of music/video from browsers and apps (Spotify, Yandex Music, etc.) through one interface.

## ✨ Features

- **Manage Windows media sessions**
- **Play/Pause**
- **Switch to next/prev one**
- **Display artist and label**
- **Display cover/preview**
- **Manage system volume**
- **Switch pin/unpin OnTop**

## 🛠️ Prerequisites

- **Windows 10/11 (build 19041 or later)**
- **.NET 8.0 Runtime**
- **The "Hardware Media Key Handling" browser feature is enabled:** To turn this on, go to search and type chrome://flags/#hardware-media-key-handling, then select Enabled in drop box.

## 🚀 Installation & Usage

### 1. Clone the Repository

```bash
git clone https://github.com/harvkl/Glass-Audio.git
cd Glass-Audio
```

### 2.  Running the Application

```bash
dotnet run
```

### Notes

> ⚠️ **Important**: If you are using Yandex Browser, you should go to `Settings -> Interface -> Show the system player when you press keys to control media` and check this box.
